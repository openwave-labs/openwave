### Drafted the core dynamic logic and time loop

```python
OBJECTS / ATTRIBUTES
BCCGranule()
- single var: mass (m, scalar)
Lattice()
- 1D array: position (p, 3D vector)
- 1D array: velocity (v, 3D vector)
Spring()
- single var: spring stiffness (k, scalar)
- single var: BCC spacing at rest (L0, 3D vector)
  - [1/2 * a, 1/2 * a, 1/2 * a]
- 1D array: index/spring-links (idx, 8D vector)
  - @initialize: indexing + mark for slicing
Timestep()
- single var: timestep (dt, scalar)
```

```python
LOOP LOGIC
every dt frame (while or for loop n frames, max 30fps? = refresh-rate limit):
- inject energy (@energy_wave.py)
  - 1 (or n) granules changed position (displacement)
  - change pos, E = F*d
compute each granule new position (@spacetime.py)
- Fs, v, x = zero # initialize local vars for safety
- compute 8-way 3D granule distance vectors (A-B)
  - x = |B-A| - L0
  - Fs = -k * x
  - a = Fs / m
- find resultant 3D force vector
- Euler Integration
  - v += a * dt # compute new velocity
  - p += v * dt # compute new position
```
