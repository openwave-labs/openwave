Relationship to Mainstream Physics
===================================

This page clarifies OpenWave's relationship to established physics.

Scientific Context
------------------

Quantum Field Theory (QFT)
~~~~~~~~~~~~~~~~~~~~~~~~~~~

**Standard Framework:**

- Experimentally validated
- Nobel Prizes (multiple)
- Predictive power established
- Foundation of particle physics

**Lattice QCD:**

- Computational method for QFT
- Discretizes spacetime on grid
- Nobel Prize 2004, 2008
- Standard for strong force calculations

Energy Wave Theory (EWT)
~~~~~~~~~~~~~~~~~~~~~~~~~

**Alternative Framework:**

- Deterministic wave mechanics
- Classical field equations
- Research/exploratory status
- Provides testable predictions

Computational Methods
---------------------

~~~~~~~~~~~~

Both use lattice discretization:

   :header-rows: 1

   * - Aspect
     - Lattice QCD
     - OpenWave Level-1
   * - Method
     - Discretize spacetime
     - Discretize spacetime
   * - Grid
     - Cubic lattice
     - Cubic lattice
   * - Fields
     - Quantum fields
     - Classical wave fields
   * - Output
     - Particle properties
     - Wave dynamics
   * - Goal
     - Solve QFT
     - Explore EWT

Differences
~~~~~~~~~~~

**Field Nature:**

- **QFT:** Quantum operators, probabilistic
- **EWT:** Classical amplitudes, deterministic

**Mathematical Framework:**

- **QFT:** Path integrals, gauge symmetry
- **EWT:** Wave equations, field gradients

**Validation Status:**

- **QFT:** Extensively validated experimentally
- **EWT:** Under computational exploration

The Medium Question
-------------------

A Common Misconception
~~~~~~~~~~~~~~~~~~~~~~

**Answer:** No. The "medium" in OpenWave is a computational construct, not a physical substance.

Computational Perspective
~~~~~~~~~~~~~~~~~~~~~~~~~

**Level-0 Granules:**

- Mathematical objects for visualization
- Like pixels in a screen
- Discretization for computation

**Level-1 Field:**

- Grid points storing field values
- Like lattice QCD grid points
- Numerical method, not ontology

**Key Insight:**

Just as lattice QCD doesn't claim spacetime is literally cubic, OpenWave uses
discretization as a numerical tool, not a physical claim.

Research Questions
~~~~~~~~~~~~~~~~~~

The research focus is:

- Can wave field dynamics reproduce observed physics?
- Do field equations match experimental data?
- Are predictions testable and falsifiable?

Physical Analogies
------------------

Not Claiming These are "True"
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

OpenWave draws computational analogies:

**Electromagnetic Waves:**

- Propagate through fields
- No medium required physically
- Field itself oscillates

**Spacetime Curvature:**

- General relativity: geometry itself
- EWT: field gradients
- Both: mathematical descriptions

**Quantum Fields:**

- QFT: quantum operators
- EWT: classical wave amplitudes
- Both: wave-field frameworks

Relationship to Experiments
---------------------------

Experimental Validation
~~~~~~~~~~~~~~~~~~~~~~~

**Current Status:**

- OpenWave: Computational exploration phase
- QFT: Extensively validated (decades of experiments)
- Goal: Compare EWT predictions to data

**Research Approach:**

1. Implement EWT computationally
2. Generate predictions
3. Compare to experimental results
4. Refine or reject based on data

Testable Predictions
~~~~~~~~~~~~~~~~~~~~

EWT aims to predict:

- Particle masses and properties
- Force strengths and ranges
- Scattering cross-sections
- Decay rates

If predictions don't match experiments, the model must be revised or abandoned.

Relationship to Other Theories
------------------------------

Pilot Wave Theory
~~~~~~~~~~~~~~~~~

**de Broglie-Bohm Mechanics:**

- Deterministic quantum theory
- Particles guided by pilot wave
- Reproduces QM predictions

**Similarities to EWT:**

- Deterministic framework
- Wave-based description
- Alternative to Copenhagen interpretation

**Differences:**

- Pilot wave: Quantum + hidden variables
- EWT: Purely classical wave fields

Wave Structure of Matter
~~~~~~~~~~~~~~~~~~~~~~~~~

**Milo Wolff's Work:**

- Standing wave models of particles
- Inspired EWT visualization

**OpenWave's Approach:**

- Computational implementation
- Quantitative predictions
- Testable against experiments

String Theory
~~~~~~~~~~~~~

**Similarities:**

- Alternative to QFT framework
- Seeks unified description
- Uses mathematical structures

**Differences:**

- String theory: Extra dimensions, supersymmetry
- EWT: 3D wave fields, classical dynamics
- Both: Research/exploratory status

Scientific Method
-----------------

OpenWave's Approach
~~~~~~~~~~~~~~~~~~~

1. **Model:** Implement EWT equations computationally
2. **Predict:** Generate quantitative predictions
3. **Test:** Compare to experimental data
4. **Refine:** Adjust model based on results
5. **Validate:** Require reproducible confirmation

Falsifiability
~~~~~~~~~~~~~~

EWT makes specific predictions that can be tested:

✅ If predictions match experiments → Continue research
❌ If predictions don't match → Revise or reject

This is standard scientific practice.
