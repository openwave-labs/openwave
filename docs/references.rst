Citing OpenWave
~~~~~~~~~~~~~~~

If you use OpenWave in your research, please cite:

**BibTeX:**

.. code-block:: bibtex

   @software{openwave2024,
     title = {OpenWave: Computational Physics Toolkit for Energy Wave Theory},
     author = {Griesi, Rodrigo and Yee, Jeff},
     year = {2024},
     url = {https://github.com/openwave-labs/openwave},
     version = {0.3.0}
   }

**APA:**

Griesi, R., & Yee, J. (2024). *OpenWave: Computational Physics Toolkit for Energy Wave Theory*
(Version 0.3.0) [Computer software]. https://github.com/openwave-labs/openwave
