Changelog

All notable changes to OpenWave will be documented here.

Version 0.3.0 (Current)
-----------------------

**Release Date:** 2024-10-XX

**Added:**

- Documentation system with Sphinx
- Comprehensive user guides
- API reference documentation
- Theory and background sections

**Changed:**

- Improved docstrings across codebase
- Updated README with clearer structure

**Status:**

- Level-0 (Granule-based): Stable
- Level-1 (Field-based): In development

Version 0.2.0
-------------

**Release Date:** 2024-XX-XX

**Added:**

- Simple Cubic (SC) lattice support
- Multiple color themes for visualization
- Asymmetric universe dimensions
- Enhanced xperiments module

**Improved:**

- Performance optimizations
- GPU memory efficiency
- Documentation updates

Version 0.1.0
-------------

**Release Date:** 2024-XX-XX

**Initial Release:**

- Body-Centered Cubic (BCC) lattice implementation
- Basic xperiments (medium vibration, spherical wave, etc.)
- CLI interface
- GPU acceleration via Taichi
- 3D visualization

**Modules Released:**

- Spacetime module (Level-0)
- Common module (constants, config, equations)
- I/O module (CLI, render)
- Validations module (wave diagnostics)
- Xperiments module (7 experiments)

Versioning
----------

OpenWave follows Semantic Versioning (SemVer):

**Format:** MAJOR.MINOR.PATCH

- **MAJOR:** Breaking API changes
- **MINOR:** New features (backward-compatible)
- **PATCH:** Bug fixes (backward-compatible)

**Pre-1.0 Status:**

Version 0.x indicates pre-release status where API may change.
